- # **隐私政策**
  **更新日期：2025年11月**

  **生效日期：2025年11月**

  感谢您使用 HiFier 应用！我们非常重视您的隐私权与个人信息保护。本隐私政策旨在清晰、透明地说明我们如何收集、使用、存储、共享和保护您的数据。

  请您在使用 HiFier 应用前，仔细阅读并理解本隐私政策。若您不同意本政策的任何内容，请停止使用我们的应用。

## 一、我们如何收集数据
  1. 应用本身
     - HiFier 应用本身 **不主动收集您的个人信息**，也不会主动收集您的设备硬件、操作系统、联系人等数据。
  2. 用户提供的信息
     - 您在使用过程中可能会提供的账号授权（如云盘挂载、媒体服务器登录信息），这些信息仅存储在您的设备本地，我们不上传、不保存、不共享。
  3. 第三方 SDK
     - 为了应用稳定运行、广告投放和统计分析，我们集成了部分第三方 SDK。这些 SDK 可能会收集与您设备相关的数据（详见[第三方 SDK 隐私相关](#三第三方sdk隐私相关)）。
## 二、我们如何使用和存储您的数据
  1. 授权信息
     - 云盘、媒体服务器的授权信息仅存储在您的设备上。删除服务或卸载应用时，该信息会被永久清除，不会保留在我们服务器上。
  2. 文件与数据
     - 应用仅以只读方式访问您挂载的云盘、媒体服务器或导入的文件。
     - 我们不会修改、上传或共享您的文件。
  3. 下载文件
     - 您在应用中下载的文件仅存储于本地，用于浏览与播放。
     - 文件可由您手动删除，卸载应用时也将一并删除，无法恢复。
  4. 播放缓存
     - 播放时自动生成的缓存文件仅用于提高播放体验。
     - 缓存文件不会上传，可在应用设置中清理。卸载应用时会一并删除，无法恢复。
  5. 播放进度
     - 若您登录媒体服务器，我们会将您的播放进度上报至该服务器，以实现记录播放历史。
     - 此功能当前无法关闭。
  6. 歌词与封面搜索
     - 默认开启时，应用会将歌曲名、艺术家名、专辑名或文件名发送至第三方搜索服务，仅用于获取歌词与封面。
     - 搜索结果仅存储在本地，不会被上传。
     - 您可在设置中关闭此功能。
  7. 听歌识曲
     - 我们调用 Apple 的系统服务来提供识曲功能。
     - 相关隐私政策请参阅：
       - [Shazam 隐私政策](https://www.shazam.com/privacy/summary)
       - [Apple 隐私政策](https://www.apple.com/privacy/)
## 三、第三方 SDK 隐私相关
1. 友盟统计 (Umeng SDK)
  - 用途：崩溃分析与统计，优化应用体验。
  - 可能收集：IMEI/MAC/Android ID/IDFA/OAID/OpenUDID/GUID/SIM IMSI/ICCID、位置信息、网络信息等。
  - 隐私政策：https://www.umeng.com/page/policy
2. 广告平台
  - **非付费用户**将会看到广告，当您订阅后，下次程序启动将不会初始化任何广告平台。
  - **中国大陆用户**：广告由穿山甲平台提供。隐私政策：https://www.csjplatform.com/privacy/partner
  - **非中国大陆用户**：广告由 Google 提供: [Google 隐私政策](https://policies.google.com/privacy)
## 四、数据的使用目的
我们收集或处理的数据仅用于：
1. 提供应用核心功能（文件播放、缓存等）。
2. 修复错误、提升稳定性、优化用户体验。
3. 分析崩溃报告，改善产品。
4. 广告个性化与投放。
## 五、数据安全措施
  1. 本地存储
     - 授权信息、缓存、下载文件均存储于本地，不会上传。
  2. 传输安全
     - 第三方 SDK 使用加密协议（如 HTTPS/TLS）传输数据。
  3. 内部管理
     - 我们不设立服务器端数据库，也不会集中存储用户隐私信息。
## 六、数据保留与删除
  1. 本地数据：您删除相关服务或卸载应用时，数据将自动删除。
  2. 第三方数据：由 SDK 提供商独立存储和管理，数据保留时间取决于其政策。
  3. 删除请求：您可通过联系我们，要求删除第三方 SDK 收集的数据。
## 七、用户权利
  根据适用的法律（如 GDPR、CCPA、中国《个人信息保护法》），您享有以下权利：
  1. **知情权**：了解我们如何收集、使用和共享数据。
  2. **访问权**：请求访问我们收集的与您相关的数据。
  3. **更正权**：请求更正不准确或不完整的数据。
  4. **删除权**：请求删除您的数据（受法律要求限制）。
  5. **限制处理权**：要求限制对您的数据的处理。
  6. **数据可携权**：请求以结构化格式导出您的数据。
  7. **撤回同意权**：在允许的情况下，撤回您对特定数据处理的同意。
## 八、未成年人保护
  - 我们不会有意收集 14 岁以下未成年人的个人信息。
  - 若监护人认为我们错误收集了未成年人的信息，请立即联系我们，我们会尽快删除。
## 九、国际用户与跨境数据传输
  - HiFier 应用可能由不同地区的用户使用。
  - 第三方 SDK（如 Google、AppLovin）可能会将数据传输至境外服务器（包括但不限于美国）。
  - 使用本应用即表示您同意可能发生的数据跨境传输。
## 十、隐私政策更新
  - 我们可能会根据业务调整、法律法规更新隐私政策。
  - 更新后的政策将在应用内展示，并标注最新的“更新日期”。
  - 若涉及重大变更，我们会在应用内进行明显提示。
